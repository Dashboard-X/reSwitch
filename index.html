reSwitch v1.0
A jQuery plugin for styling checkboxes and radio buttons

The need for customizing every aspect of the forms we designed has driven us crazy searching different plugins to help us customize checkboxes and radio groups. There is a lot of good and bad out there. It just depends on how much each and every one of those out there suits you.

So we've got to this point where we wanted it all: heavy css, animation and responsiveness. Doodling around with the code gave birth to this plugin we call reSwitch.

How to style checkboxes with css and javascript using the reSwitch jQuery plugin

Checkboxes

The HTML

<label>Wireless</label>
<input type="checkbox" name="settings" value="wireless" checked>
<label>Bluetooth</label>
<input type="checkbox" name="settings" value="bluetooth" checked>
<label>Data</label>
<input type="checkbox" name="settings" value="data">

The JavaScript

$('[type="checkbox"]').reswitch({
	textOn: 'I',
	textOff: 'O',
	hasToggle: true,
	theme: 'iphone4'
});

Styling radio buttons with reSwitch

Radio groups

The HTML

<label>General</label>
<input type="checkbox" name="profile" value="general" checked>
<label>Meeting</label>
<input type="checkbox" name="profile" value="meeting">
<label>Silent</label>
<input type="checkbox" name="profile" value="silent">

The JavaScript

$('[type="radio"]').reswitch({
	textOn: 'I',
	textOff: 'O',
	hasToggle: true,
	theme: 'iphone4 orange'
});
