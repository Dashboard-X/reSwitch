reSwitch v1.0

reSwitch is a jQuery plugin for styling checkboxes and radio buttons

There are a lot of good and bad jQuery plugins in the wild when it comes to replacing the browsers' default checkboxes and radio buttons. reSwitch may just be what

Download View project on GitHub

How to style checkboxes with css and javascript using the reSwitch jQuery plugin

Checkboxes

The HTML

<label>Wireless</label>
<input type="checkbox" name="settings" value="wireless" checked>
<label>Bluetooth</label>
<input type="checkbox" name="settings" value="bluetooth" checked>
<label>Data</label>
<input type="checkbox" name="settings" value="data">

The JavaScript

$('[type="checkbox"]').reswitch({
	textOn: 'I',
	textOff: 'O',
	hasToggle: true,
	theme: 'iphone4'
});

Styling radio buttons with reSwitch

Radio groups

The HTML

<label>General</label>
<input type="checkbox" name="profile" value="general" checked>
<label>Meeting</label>
<input type="checkbox" name="profile" value="meeting">
<label>Silent</label>
<input type="checkbox" name="profile" value="silent">

The JavaScript

$('[type="radio"]').reswitch({
	textOn: 'I',
	textOff: 'O',
	hasToggle: true,
	theme: 'iphone4 orange'
});
